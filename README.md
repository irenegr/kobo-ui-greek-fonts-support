# kobo-ui-greek-fonts-support
Support for Greek fonts in names, titles, collections and dictionary definitions.

<b>Installation:</b>
1. Connect the device to pc
2. Copy the KoboRoot.tgz to (hidden) .kobo folder
3. Safely eject the device and <b>wait</b> to reboot
4. Enjoy

<u>Testing device</u>: Kobo Clara BW, firmware: 4.42.23296	May 2025
