# kobo-ui-greek-fonts-support
Support for Greek fonts in names, titles, collections and dictionary definitions for Kobo Clara BW.

<b>Installation:</b>
1. Connect the device to pc
2. Copy the KoboRoot.tgz to (hidden) .kobo folder
3. Safely eject the device and <b>wait</b> to reboot
4. Enjoy

<i>Testing device</i>: <b>Kobo Clara BW</b>, firmware: 4.42.23296	May 2025
